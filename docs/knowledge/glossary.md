# Глоссарий доменных терминов

## Бизнес

| Термин | Расшифровка |
|--------|-----------|
| **ИЗИБУК / EasyBook** | Компания по аренде мероприятий / фотобудок |
| **ИЗИНЕОН / EasyNeon** | Компания по производству неоновых вывесок |
| **IZIBOOK** | Enum-значение Company в Prisma |
| **IZINEON** | Enum-значение Company в Prisma |
| **ДРР** | Доля рекламного расхода = spend / revenue × 100% |
| **Заявка / CPL** | Cost Per Lead — стоимость одного лида (заявки) |
| **Кампания** | Рекламная кампания в VK Ads (campaignId + campaignName) |
| **Снимок** | `AdSnapshot` — накопленные за день показатели кампании на момент съёмки |
| **Дельта** | Разница между двумя снимками — реальный расход/показы/клики/лиды за час |
| **Кабинет** | `VkCabinet` — рекламный кабинет VK Ads (один аккаунт в ads.vk.com) |
| **Постоянник** | TBD — термин из бизнес-контекста EasyBook/EasyNeon |

## Метрики VK Ads

| Метрика | Формула | Описание |
|---------|---------|---------|
| **CPL** | spend / leads | Стоимость лида |
| **CPM** | spend / impressions × 1000 | Стоимость 1000 показов |
| **CPC** | spend / clicks | Стоимость клика |
| **Предварительные данные** | `isPreliminary = true` | VK корректирует данные последних ~2 ч |
| **Полуночный сброс** | `end.spend < baseline.spend` | VK обнуляет счётчики в 21:00 UTC |

## Мессенджер

| Термин | Описание |
|--------|---------|
| **Гендир** | GEN_DIRECTOR — роль мессенджера генерального директора |
| **КомДир** | COMMERCIAL_DIRECTOR — выдаёт гранты доступа к Гендиру |
| **Грант** | `GendirAccessGrant` — разрешение менеджеру/дизайнеру писать Гендиру напрямую |
| **LEADERSHIP** | Массив ролей с расширенными правами: [GEN_DIRECTOR, COMMERCIAL_DIRECTOR, SALES_DIRECTOR, ROP, DESIGN_DIRECTOR, PRODUCTION_HEAD] |
| **Profile gate** | Модальное окно с заполнением профиля при первом входе в мессенджер |
| **Test mode** | `MESSENGER_TEST_MODE=true` — матрица прав отключена |
| **NEWS** | ChatType.NEWS — канал новостей, только руководство пишет |
